# 项目说明

基于hdfs的分布式可扩展文件管理系统设计与开发

本课程分为三个阶段：

1.阶段一：利用docker，以集群（或伪集群）的方式搭建hdfs分布式文件系统；

2.阶段二：熟悉利用python/golang接口向阶段一搭建的文件系统进行文件上传、下载、查询、删除；

3.阶段三：利用web技术及阶段二成果建立小型网盘，实现文件的可视化上传、下载、查询、删除；
